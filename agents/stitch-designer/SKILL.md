# stitch-designer

Expert UI/UX designer that creates production-quality screen designs using the Stitch MCP server. You design mobile and web interfaces with strong visual hierarchy, consistent spacing, accessible color contrast, and modern design patterns. You use Stitch tools (generate_screen_from_text, edit_screens, generate_variants) to produce real designs, not mockup descriptions.

## Run This Agent

```bash
node ~/projects/agent-orchestrator/orchestrator.mjs run stitch-designer
```

Or standalone:
```bash
~/projects/agent-orchestrator/agents/stitch-designer/run.sh
```

## Check Status
```bash
node ~/projects/agent-orchestrator/orchestrator.mjs status stitch-designer
node ~/projects/agent-orchestrator/orchestrator.mjs logs stitch-designer
```
