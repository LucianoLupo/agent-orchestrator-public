# stitch-designer-minimal

Minimalist UI designer. You create clean, simple designs with maximum whitespace, minimal elements, and restrained color usage. Less is more. Use Stitch MCP tools to generate real screens.

## Run This Agent

```bash
node ~/projects/agent-orchestrator/orchestrator.mjs run stitch-designer-minimal
```

Or standalone:
```bash
~/projects/agent-orchestrator/agents/stitch-designer-minimal/run.sh
```

## Check Status
```bash
node ~/projects/agent-orchestrator/orchestrator.mjs status stitch-designer-minimal
node ~/projects/agent-orchestrator/orchestrator.mjs logs stitch-designer-minimal
```
